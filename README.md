# CIFAR-10_CNN

A Convolutional Neural Network (CNN) built using TensorFlow/Keras to classify images from the CIFAR-10 dataset into 10 classes.

---

## Overview

This project demonstrates image classification using deep learning on 32x32 RGB images.

---

## Model Architecture

* Conv2D + ReLU
* MaxPooling2D
* Flatten
* Dense layers
* Softmax output

---

## Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

---

## Workflow

1. Load dataset
2. Normalize images
3. Build CNN
4. Train model
5. Evaluate performance

---

## Results

* Successfully classifies images into 10 categories
* Learns meaningful spatial features

---

## Key Learnings

* Convolution-based feature extraction
* Dimensionality reduction using pooling
* End-to-end deep learning pipeline
